# village-emrg — чеклист ручных проверок

То, что не покрывается unit/integration/E2E и требует реального устройства,
браузера или внешнего сервиса. Заполняется по мере реализации фронта (Plan 2)
и деплоя (Plan 3).

## Медиа: компрессия фото (Task 07)

Канвас и Web Worker не исполняются в jsdom — проверяется вручную в браузере:

- [ ] Загрузка фото с камеры/галереи (JPEG, PNG, HEIC где поддерживается).
- [ ] Результат — WebP, длинная сторона ≤ 1920 px.
- [ ] Размер после компрессии ≤ 1 МБ для типового фото с телефона.
- [ ] EXIF (включая GPS) отсутствует в результате (canvas отбрасывает метаданные).
- [ ] До 5 фото на инцидент; шестое не добавляется.
- [ ] Компрессия идёт в Worker — UI не подвисает на больших файлах.
